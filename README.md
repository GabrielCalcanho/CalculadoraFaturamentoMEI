💰 Calculadora de Faturamento MEI

Aplicação web desenvolvida para ajudar microempreendedores (MEI) a controlarem seu faturamento mensal e anual de forma simples e eficiente.

🚀 Funcionalidades

✅ Registro de vendas diárias

📊 Cálculo automático do faturamento mensal

💸 Exibição do limite anual restante (MEI)

📋 Listagem das vendas adicionadas

🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript 


📚 Aprendizados

Este projeto foi desenvolvido com foco em:

Manipulação do DOM

Eventos em JavaScript

Organização de código

Lógica de programação aplicada ao mundo real

📈 Gráficos de faturamento

📤 Exportação de relatórios (PDF)

☁️ Integração com banco de dados
